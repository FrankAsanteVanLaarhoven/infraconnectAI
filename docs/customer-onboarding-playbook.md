# VLA Workbench — Customer Onboarding Playbook

## Phase 1: Discovery (Week 1)
- Technical discovery call (current stack, pain points)
- Security & compliance review
- Define success metrics (GPU hours saved, iteration speed)

## Phase 2: Pilot (Weeks 2–4)
- Deploy sandbox environment
- Seed with customer’s real datasets (BDD100K / LIBERO)
- 2-week pilot with 3 engineers

## Phase 3: Production Rollout (Week 5+)
- Full Kubernetes deployment
- SSO + RBAC integration
- Training for team (2× 90-min sessions)
- Go-live support (first 30 days)

## Success Metrics
- >40% reduction in wasted GPU hours
- >3× faster model iteration
- 100% audit compliance
