1. Click **New Run** → Launch with 1024 environments
2. Switch to **Analytics** tab → Show live updating charts
3. Click an episode → Open 3D replay modal
4. Prune a low-quality episode live

## Minute 3–5: Real-time & Collaboration
- Open the same workspace in a second tab
- Show live episode updates appearing in real time
- Highlight the green connection dot

## Minute 5–7: ROS2 & Physical Robot Support

- Show live joint states + battery + anomaly detection
- Open **RViz View** tab → Demonstrate 3D visualization

## Minute 7–9: URDF Import & Real-to-Sim
- Click **Import Robot (URDF)**
- Drag & drop a real robot URDF
- Show auto-sync to simulation with exact real-world specs
